## 来源 1：店小秘官方帮助中心

来源：<https://help.dianxiaomi.com/pre/getContent.htm?id=2346>

### 商品标题
- 长度：**≤ 250 字符**
- 语言：**建议英文**
- 功能：支持「首字母大写」
- 内容：简洁扼要描述产品核心卖点、功能

### 产品描述
- 说明书：仅 PDF，≤ 15M，需含英文说明
- 描述编辑：支持编辑文字 + 上传图片

### 图片限制

| 类型 | 数量 | 尺寸 | 比例 | 文件大小 |
|------|------|------|------|---------|
| 轮播图/主图 | ≤ 10 张 | ≥ 800×800 | 1:1 | ≤ 2M |
| 产品素材图 | 自动取第一变种第一张 | ≥ 800×800 | 1:1 | - |
| 外包装图片 | 1-6 张（正/侧/俯拍） | - | - | - |
| 视频 | - | - | 1:1 / 3:4 / 16:9 | ≤ 500M，≤ 1 分钟 |

### 必填字段
- 产品属性（带 *）
- 产品分类
- 服装/鞋类尺码表
- 颜色/尺码变种主题

### 其他硬约束
- **字符及标点必须用英文输入法 → 否则发布失败**
- 多变种：**变种个数 ≤ 20**
- **SKU 货号：不能含中文和中文符号**
- 在线产品：不可编辑，只能查看

## 来源 2：WebSearch 综合（公开文章）

### 标题
- 全英文
- 多件装须标明（pcs / pack / set / kit / pieces / bundle）
- 壳膜类需标明 `for iPhone` / `for Samsung`

### 描述
- 最多 50 个模块，可为空或至少 1 个图片模块
- **单文字模块字符数 < 500**
- 描述总长度：2000 字符（一说 5000，不同来源不一致）

### 违禁词与合规
- 不得含「免费」「秒杀」等夸大词
- 不得引导评价（不能用激励换好评）
- 店名/品牌名不能含夸张表达、激励性内容
- 店名不能含国家、地区、国际组织名称
- 不得涉及侵权、违法、与商品无关的内容

### 图片（与店小秘官方有冲突）
- 公开文章：0.5 ≤ 宽/高 ≤ 2、宽 ≥ 480、高 ≥ 480
- 店小秘官方：比例 1:1、≥ 800×800
- **决策**：取严格一方（店小秘官方）

## 未落地的规则（备忘）

- **描述总长度**：来源数据冲突（2000 vs 5000），需澄清
- **单文字模块 < 500**：需要点开「编辑描述」模态才能识别每个文字模块
- **描述模块数 ≤ 50**：同上
- **轮播图数量 ≤ 10 / 文件大小 ≤ 2M**：数量易加（但用户实际不会超）；文件大小在 DOM 上拿不到
- **视频规则**：暂不检查视频字段
- **说明书 PDF 规则**：暂不检查说明书字段
- **店名/品牌名规则**：发布页无此字段
- **壳膜类专项规则（`for iPhone` 必须）**：需要类目识别，类目字段在当前页未识别
- **外包装图片 1-6 张数量限制**：与 `required_fields_empty` 部分重复（缺图会被必填检查捕获，但「3 张」也会通过 — 数量上限未单独检查）
